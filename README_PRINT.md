# resume_print.html 使用说明

## 快速生成PDF的方法

### 方法一：浏览器打印（推荐）

1. 双击打开 `resume_print.html` 文件
2. 在浏览器中按 `Ctrl + P`（Windows）或 `Cmd + P`（Mac）
3. 在打印机选择中选择 **"另存为PDF"**
4. 点击"保存"或"打印"按钮

**打印设置优化：**
- 纸张大小：A4
- 边距：默认或自定义（建议1.5cm）
- 背景图形：勾选"打印背景"
- 缩放：100%

### 方法二：直接生成PDF（需要Python环境）

如果您已经安装了reportlab库，可以运行：

```bash
python generate_pdf.py
```

这将自动生成 `陶诗曼_简历_2026.pdf` 文件。
